# 싯딤 Shittim

싯딤(Shittim)은 **넥슨게임즈**에서 개발한 모바일 게임 **블루아카이브**의 IP를 활용한 히어로 슈터 장르의 2차 창작 팬게임입니다.  
플레이어는 **블루아카이브**에 등장하는 캐릭터를 직접 조작하며 다른 플레이어와의 전투(PVP), 스토리 상의 적들과 전투(PVE)를 진행합니다.

- **장르**: 히어로 슈터 (FPS / TPS)
- **플랫폼**: Windows, Anroid
- **게임 플레이**
  - 데스매치: 상대방 팀의 모든 인원이 전투 불능 상태가 되면 승리합니다.
  - 점령전: 자신의 팀 점령도가 상대팀보다 먼저 100%에 도달하면 승리합니다. 점령지 내에 각 팀이 존재할 경우 각 팀의 점령도가 상승하며, 양 팀이 모두 점령지 내에 존재하는 경우 점령도가 상승하지 않습니다. 점령지 내 팀 인원이 많을수록 점령도 상승 속도가 빨라집니다.
  - 총력전: 강력한 보스와 전투합니다. 클리어한 시간이 랭킹에 기록됩니다.
  - 통공의 고성당: 몰려오는 유스티나 성도회를 상대로 최대한 오래 버티는 것이 목표입니다. 버틴 시간이 랭킹에 기록됩니다.
- **도움말**
  - 학생 선택 화면에서 학생을 선택한 후 우상단에 표시되는 이름을 클릭하거나 키보드 F1키를 누르면 정보가 표시됩니다.
  - 일정 시간 동안 피해를 받지 않으면 체력이 회복됩니다.
  - 일정 시간마다 스페셜 스킬을 사용할 수 있습니다.
  - 적을 공격하여 EX 스킬 발동에 필요한 코스트를 회복할 수 있습니다.
  - 코스트를 소모하여 체력을 회복할 수 있습니다.
- **조작법**
  - 이동: W, A, S, D
  - 점프: 스페이스바
  - 슬라이딩: LShift
  - 기본 공격: 좌클릭
  - 조준(견착): 우클릭
  - 재장전: R
  - 스폐셜 스킬: Q
  - EX 스킬: E
  - 체력 회복(코스트 소모): T
  - 행동 캔슬: C
  - 시점 전환: V
  - 자유 시점: Alt [홀드]
  - 채팅: Enter
